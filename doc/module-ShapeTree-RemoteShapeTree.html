Class: RemoteShapeTree

ShapeTree~RemoteShapeTree(url:, path:)

reference to a ShapeTree stored remotely

Constructor

new RemoteShapeTree(url:, path:)

Parameters:
Name Type Description
url:

URL string locating ShapeTree
path: labelName

refer to a specific node in the ShapeTree hierarchy

A ShapeTree has contents: [] a rdf:ShapeTreeRoot, ldp:BasicContainer ; tree:contents

The contents may be ldp:Resources: [ a ldp:Resource ; tree:uriTemplate ".ttl" ; tree:shape gh:LabelShape ] ], or ldp:Containers, which may either have n nested static directories: [ a ldp:BasicContainer ; rdfs:label "repos" ; tree:contents ... ] , [ a ldp:BasicContainer ; rdfs:label "users" ; tree:contents ... ] or one dynamically-named member: [ a ldp:BasicContainer ; tree:uriTemplate "{userName}" ; tree:shape gh:PersonShape ; tree:contents ]
